## What to build

Hand-compose one finished botanical plate in Figma, Affinity, or similar tool. Use real outputs from phase 0.1a (identification JSON) and phase 0.1b (FLUX illustration). The goal is to confirm the typographic layout works at the target export size before any SwiftUI Canvas code is written.

## What to compose

- Aged paper texture background (use real vintage paper texture asset)
- FLUX illustration composited
- Title: common name
- Scientific name (italic)
- Family
- Notes panel (placeholder notes text)
- Border

Single botanical portrait layout. No anatomical labels or leader lines in v0.1 (FLUX outputs a flat image without internal geometry).

## Acceptance criteria

- [ ] Composed plate reads as a coherent botanical plate
- [ ] Title, scientific name, family, illustration, notes panel, paper texture, border all sit together without fighting each other
- [ ] Typographic hierarchy is clear and print-quality
- [ ] Export size is confirmed (target: 2480 x 3508 px at 300dpi, or similar portrait print format)
